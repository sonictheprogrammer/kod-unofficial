### NetBSD

To install git use pkg.

`pkg_add -v git`

Then, Clone the project.

`git clone https://github.com/sonictheprogrammer/kod-unofficial`

Make it into a cd.

`cd kod-unofficial`

Unlock it. 

`chmod +x shell.sh`

Run it by using this command,

`./shell.sh`
